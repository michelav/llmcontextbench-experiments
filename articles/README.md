# Papers

This directory will hold the camera-ready PDFs of the two SBES 2026 papers associated with this replication package:

- *[LLMContextBench: A Benchmark Tool for Evaluating Context Provisioning Strategies in LLM-Based Systems](llmcontextbench-tool-track.pdf)*;
- *[Evaluating Context Provisioning Strategies for LLM-Based Systems: An Empirical Study with the Lattes Platform](llmcontextbench-research-track.pdf)*.

They will be added here before the Festival de Artefatos submission deadline, once the camera-ready versions are available. This satisfies the CBSoft requirement that the README link to the accepted paper, since the PDF can live inside the artifact repository itself.
